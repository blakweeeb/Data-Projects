## 💡 Key Focus Areas

### 1. Data Engineering & Scalability
Proyectos enfocados en la arquitectura distribuida. Uso de **PySpark** para el procesamiento de volúmenes masivos de datos (TB/PB), optimización de esquemas y formatos de almacenamiento como **Parquet**.


### 2. Business Intelligence & Automation
Implementación de soluciones de extremo a extremo para automatizar flujos de trabajo (ETLs sin código con **KNIME** o **n8n**) y visualización de datos críticos para la toma de decisiones empresariales usando **PowerBI** y **Tableau**.


### 3. Backend & API Services
Desarrollo de microservicios y APIs robustas. Diseño de arquitecturas limpias, autenticación y despliegue de soluciones escalables.
